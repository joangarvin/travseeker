# utils

Funciones puras de ayuda sin dependencias de React.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `parseJson.ts` | `parseJsonSafe`: parsea strings JSON del backend (ubicación, presupuesto, etc.) con fallback al string original si falla. |
| `images.ts` | `getImageUrl`: resuelve la URL de imagen de un destino, con fallback a placeholders de `constants/images.ts`. |
| `masification.ts` | Utilidades para etiquetas y colores del nivel de masificación turística. |
| `admin/` | Conversión de datos del panel admin. Ver [admin/](./admin/). |
