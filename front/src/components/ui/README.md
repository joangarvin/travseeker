# ui

Componentes de interfaz genéricos, reutilizables en cualquier página.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `ThemeToggle.tsx` | Botón para alternar entre modo claro y oscuro. |
| `ScrollReveal.tsx` | Wrapper que anima la entrada de elementos al hacer scroll. Acepta delay 0–3. |
| `PageLoader.tsx` | Pantalla de carga a pantalla completa con spinner y texto opcional. |
| `LoadingSkeleton.tsx` | Placeholders animados para tarjetas de destino mientras cargan datos. |
| `ConnectionError.tsx` | Banner de aviso cuando el frontend no puede conectar con el backend. |
