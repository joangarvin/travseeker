# assets

Imágenes y recursos gráficos importados directamente en componentes con `import logo from '../assets/logo.png'`. Vite las optimiza y les asigna hash en producción.

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `logo.png` | Logo de Travseeker usado en Header, About y fondos decorativos. Copia de `Logo/LOGO FINAL.png`. |
| `hero.png` | Imagen de hero (legacy; el diseño actual usa gradientes en lugar de foto de stock). |
| `react.svg` | Icono de React del template inicial de Vite. |
| `vite.svg` | Icono de Vite del template inicial. |
