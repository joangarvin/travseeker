# public

Recursos estáticos que Vite copia tal cual al build. Se acceden con rutas absolutas desde la raíz (`/favicon.svg`, `/fonts/...`).

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `favicon.svg` | Icono de la pestaña del navegador. |
| `icons.svg` | Sprite o conjunto de iconos SVG reutilizables. |

## Carpetas

| Carpeta | Descripción |
|---------|-------------|
| `fonts/` | Fuentes tipográficas de la marca. [README](./fonts/README.md) |
